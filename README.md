# 🚀 Learn-FullStack-90Days

> **A public learning journal** — 90 days of Python, SQL, and Frontend, built in the open for fellow students.

Inspired by [30-Days-Of-Python](https://github.com/asabeneh/30-days-of-python), this repo documents my full-stack learning journey from zero to job-ready.

---

## 🗺️ What I'm Learning (3 Tracks Every Day)

| Track | Topics |
|-------|--------|
| 🐍 **Python** | Basics → OOP → Advanced → Django → DRF |
| 🗄️ **SQL** | MySQL basics → JOINs → Window functions → Django ORM |
| 🎨 **Frontend** | HTML → CSS → JavaScript → Fetch API → CRUD UI |

---

## 📅 Roadmap Overview

### Month 1 — Foundations (Days 1–30)
Python basics, SQL basics, HTML & CSS fundamentals. End goal: Personal portfolio live on GitHub Pages.

### Month 2 — Building (Days 31–60)
Intermediate Python, Django intro, JavaScript. End goal: Django blog app deployed with frontend.

### Month 3 — Production (Days 61–90)
Django REST Framework, JWT auth, deployment on Railway, Capstone project, Interview prep.
